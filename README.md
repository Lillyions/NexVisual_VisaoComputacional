<h2 id="about">📌 About</h2>

This repository brings together practical activities developed during my preparation in Computer Vision.

The notebooks explore the main stages of a computer vision workflow, including image loading, preprocessing, feature extraction, model inference, model fine-tuning, result visualization and experiment tracking.

Rather than being a single final project, this repository works as a learning portfolio with applied exercises and experiments in visual artificial intelligence.

<h2 id="content">📚 Course Content</h2>

<h3>Module 1 - Fundamentals</h3>

- NumPy array manipulation
- Image visualization
- Introduction to OpenCV
- Basic image operations
- Spatial filters and convolutions

<h3>Module 2 - Features and Pre-Trained Models</h3>

- Feature detection and extraction
- Feature matching between images
- Use of pre-trained models with PyTorch and TorchVision

<h3>Module 3 - Applied Deep Learning</h3>

- Transfer learning

- Semantic segmentation

<h3>Module 4 - Generative and Multimodal Models</h3>

- Generative Adversarial Networks, GANs
- Fundamentals of multimodal models
- Image and text association with CLIP

<h3>Module 5 - Practical Applications</h3>

- Visual inspection of items on a manufacturing conveyor
- Text recognition in images with OCR
- Identification of visual elements in mobile app interfaces
- Surveillance system with computer vision
- Segmentation of defects in fabrics
- Lane detection for autonomous vehicles

<h3>Module 6 - Experiment Monitoring</h3>

- Experiment tracking
- TensorBoard logs
- Training metrics organization

<h2 id="started">🚀 Getting Started</h2>

<h3>Prerequisites</h3>

To run the notebooks, you will need:

- [Python](https://www.python.org/)
- [Jupyter Notebook](https://jupyter.org/)
- [GitHub Account](https://github.com/)

<h3>Cloning</h3>

Clone this repository using the web URL:

```bash
https://github.com/Lillyions/NexVisual_VisaoComputacional.git
```

<h3>Installing dependencies</h3>

Some notebooks require additional libraries. You can install the main dependencies with:

```bash
pip install numpy matplotlib opencv-python torch torchvision ultralytics easyocr datasets tqdm huggingface_hub tensorboard
```

<h3>Running the notebooks</h3>

Start Jupyter Notebook:

```bash
jupyter notebook
```

Then open the desired notebook and run the cells.

Some activities may download datasets or pre-trained models during execution, so an internet connection may be required.
